# Zentry - The Metagame Layer

## üöÄ Introduction
Zentry is a cutting-edge gaming website that bridges the gap between Web2 and Web3 gaming. It serves as a unified Play Economy, offering a shared adventure where players can discover, play, and earn across various games and platforms.

## ‚ú® Features
- **Immersive Hero Section**: A dynamic video-background hero section with GSAP animations and a "redefine" theme.
- **Bento Grid Layout**: A visually stunning features section using a tilt-effect Bento grid to showcase key components like "Radiant", "Zigma", "Nexus", and "Azul".
- **Interactive About Section**: A scroll-triggered image reveal animation that introduces the "Game of Games" concept.
- **Story-Driven Experience**: Designed to immerse users in a narrative-rich environment.
- **Responsive Design**: Fully optimized for various screen sizes.

## üì¶ Installation & Usage
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd zentry-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
